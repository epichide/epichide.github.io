概率论与随机过程笔记
2020-08-06| math|

基本概念

  1. 随机试验(试验) experiment E :
    1. 可重复性 : 在相同条件下可以重复
    2. 结果可知性 : 不能知道准确的结果,但事先知道所有可能的结果
  2. 样本点 sample point ω :
    • 随机试验每一个不可分割的结果
  3. 样本空间 sample space Ω : ω ∈ Ω
    • 样本点全体的集合
      1. 离散样本空间
        1. 有限样本空间
        2. 可列样本空间
      2. 连续样本空间
  4. 随机事件 event A : A ⊂ Ω
    • 样本空间子集
    • 表示 : {ω | X(ω) ≤ x } = {X≤x} = X≤x , A = {ω |X(ω) ∈ A } = { X(ω) ∈ A } = X ∈ A
    • 运算 : 余 A^c^ , $\bar{A}$ , 对称差 (仅其一发生) A △ B ,
      1. 互斥事件 AB=∅
      2. 对立事件/逆事件
  5. 随机变量 random variable X(ω) , X , ξ , ζ , η
    • 定义在样本空间的实值函数
  6. 极限事件
    1. 上极限事件

概率公式

  1. 概率加法Jordan公式
    $$P(A_1\cup A_2…\cup A_n)=\sum_{j=1}P(A_i)-\sum_{1≤i<j≤n}P(A_iA_j)+\ldots +(-1)^{n-1}\sum_{1≤j_1<j_2\ldots <j_n}P(A_{j_1}A_{j_2}\ldots A_{j-n})$$
  2. 概率乘法公式
    $$P(A_1A_2…A_n)=P(A_n|A_1 A_2…A_{n-1})P(A_{n-1}|A_1A_2…A_{n-2})\ldots P(A_2|A_1)P(A_1)$$
  3. 全概率公式
    • $B_iB_j=\varnothing, i \neq j$
    • $A⊂B_1\cup B_2\cup…\cup B_n$
      $P(A) = P(A|B_1)P(B_1) + P(A|B_2)P(B_2) + … + P(A|B_n)P(B_n)$$
  4. Bayes贝叶斯公式
    $$P(B_i|A) = \frac{P(A|B_i)P(B_i)}{\sum_{j=1}^{n} P(A|B_j)P(B_j)}$$
