  • 装箱 boxing (基本类型转换为基本类型的类对象)
  • 自动装箱：将 基本数据类型 封装为对象类型，来符合java的面向对象的思想。
  • 自动拆箱：将对象重新转化为基本数据类型。
  • 声明类型 declear type (一个变量声明为某种类型,一个引用变量声明为某种类的引用)
  • 实际类型 actual type (被应用变量的实际类型/类)
  • 对象转换
  • 隐式转换 implict casting (子类是实例被引用给父类引用变量)
  • 向上转换 upcasting
  • 显示转换 explict casting (子类 b =(子类)父类对象,书写方式,把父类转换为子类)
  • 向下转换 downcasting

  • 可访问性修饰符 accessibility modifier
